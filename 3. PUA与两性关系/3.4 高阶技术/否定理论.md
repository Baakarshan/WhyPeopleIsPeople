# 否定理论

**核心一句话总结**：适度不迎合能抬高你的价值，纯否定只会让你变成垃圾人。 

## 1. 底层原理

PUA 里的「否定」本来是指：  
- 不一味顺从对方，而是敢表达自己的标准与不满  

健康版本：  
- 自尊 + 边界 → 不迎合一切  

病态版本：  
- 通过贬低、打压、羞辱对方，建立虚假的优势地位  

后者实质是情感暴力。

## 2. 现实中的残酷表现

1）用玩笑不断讽刺对方外貌、能力、社交，引发对方自卑。  

2）在公共场合否定对方，让TA在众人面前下不来台。  

3）一有情绪就用否定攻击对方：「你就这个水平」「难怪你没人要」。  

4）有的人以为这叫「高价值框架」，其实只是人格扭曲。

## 3. 操控/防御/利用方法

1）用「自我肯定」替代对他人否定来抬高自己：  
-表达你的标准，而不是踩低别人来证明你有标准。  

2）敢拒绝、不迎合，但不羞辱：  
-「这件事我不太喜欢」「这样的相处方式不适合我」，不需要变成「你有病」「你就是垃圾」。  

3）防御对方对你使用病态否定：  
-如果对方长期用打压、冷嘲热讽来维持自尊，这人不值得你待。  

4）及时指出对方的否定对你的影响：  
-「你这样的说话方式让我觉得被贬低，如果继续这样，我会选择离开。」  

5）用否定行为不否定人格：  
-可以说「这种做法很糟」，但少用「你这个人就是××」。  

6）少用阴阳怪气：  
-不直接说问题，只冷讽。长期只会把关系腐蚀掉。

## 4. 常见陷阱与反噬

1）误学到「要吸引就要不停否定对方」：  
-你学了一点技巧，却不懂人性底线。  

2）用否定来掩饰自己的不安全感：  
-你怕被看低，就先把对方打低。  

3）一味迁就别人否定自己：  
-你把这种打压当成「高价值」，让自己越陷越低。  

4）把「真话」和恶意否定混为一谈：  
-你可以讲真话，但有讲法。  

5）认为靠否定维持的关系能长久：  
-实际上对方早在心里离开你，只是在等一个机会。

## 5. 进阶阅读与验证

- 书籍：沟通、非暴力沟通相关。  
- 书籍：情感虐待、PUA 防御相关书籍。  
- 论文方向：搜索「verbal aggression relationships」「psychological abuse」。  

自我验证：  
1）列出你最近对伴侣/暧昧对象说的三句「否定式话语」。  
2）问自己：这是在表达边界，还是单纯泄愤？  
3）下一次想否定时，换成「说事实+说感受+说需求」的格式试一次。  
4）你会发现，你可以不跪，也可以不变成垃圾。
