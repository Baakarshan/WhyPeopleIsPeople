# 框架重塑

**核心一句话总结**：你不是没有框架，是你的旧框架把你搞成现在这样。 

## 1. 底层原理

个人框架来自：  
- 原生家庭（你值不值得被爱）  
- 早期关系（你被怎样对待）  
- 自己反复对自己说的话  

这些组成了你对「我是谁」「别人应该如何对待我」「我在关系里该扮演什么」的底层设置。  

你不去重塑，它就会自动运行，让你一遍遍选同一类人，一遍遍走进同一类坑。

## 2. 现实中的残酷表现

1）「我必须证明我配得上」框架：  
你总是在关系中过度付出、讨好、委屈自己，生怕对方发现你「其实不值」。  

2）「没有人会真心对我」框架：  
你遇到真心的人也会怀疑、测试、折腾，最后把对方推走，用结果证明自己是对的。  

3）「男人就该痛苦挣钱」框架：  
你把自己熬到半死，完全不允许快乐和放松，最后身心俱疲。  

4）「我注定被抛弃」框架：  
你会下意识地选不靠谱的人、上头得更快、付出得更不计后果——好像就是为了再次验证这个信念。

## 3. 操控/防御/利用方法

1）写下你对自己最根本的三句信念：  
比如「我不值得被爱」「我必须有钱才有价值」「我天生社恐」。  

2）追溯这些信念的来源：  
是父母说的？前任行为造成的？还是你某次失败后自己编的？看清楚它们其实只是故事，不是事实。  

3）设计新的框架语句：  
例如「我值得被尊重，不需要用过度付出来换」「我允许自己既赚钱又享受」「我可以不擅长社交，但可以被喜欢」。  

4）在行为层面强行匹配新框架：  
比如旧框架让你遇到不尊重就忍，新框架要求你说不；一开始很不适，但必须干。  

5）筛选只适合新框架的人留下：  
你的旧圈子里可能很多人是以旧框架在消耗你。你要敢于远离这些人。  

6）给自己时间：  
框架重塑不是喊几句口号，是反复在关键时刻选择新的反应模式。

## 4. 常见陷阱与反噬

1）只在头脑里重塑，不在行为上动：  
你写了很多漂亮的信念，让你感觉自己好像成长了，但遇事还是老样子。  

2）把新框架当成「我要变成另一个人」：  
你不是换皮，而是把曾经的扭曲调回来一点，让自己没那么容易被踩。  

3）想在旧框架关系里重塑：  
你想在一直压迫你的人面前，练习新的边界和框架，难度极高。  

4）遇到一次挫折就觉得「我根本改不了」：  
旧框架运行多年，一两次失败很正常，你要看长期趋势。  

5）把一切问题都怪到框架上：  
该提升硬实力的地方，你不干，只在框架上打转靠嘴。

## 5. 进阶阅读与验证

- 书籍：认知疗法/自我对话相关，如《感觉良好》《重塑心灵》。  
- 书籍：依恋理论与自尊建设相关。  
- 论文方向：搜索「self-schema」「cognitive restructuring」。

自我验证：  
1）挑出一条你最想重塑的旧框架，写出新版本。  
2）在接下来一个月里，每遇到相关情境，就按新框架行事一次，哪怕很小。  
3）记录结果：真的像你旧框架预言的那样糟吗？  
4）你会发现：很多时候，是你自己旧框架在搞你，不是世界一定要那样对你。
