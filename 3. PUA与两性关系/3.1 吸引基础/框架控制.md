# 框架控制

**核心一句话总结**：要么你用自己的规则过滤人，要么你被别人的剧本和情绪牵着当道具。

## 1. 底层原理

框架=对“我们是什么关系”“游戏规则是什么”“什么是正常、什么是过线”的默认解释权。

大脑为了省力，不可能每次互动都从零谈规则，于是会自动选一个框架：
- 来自权力更大、更稳定、更清晰的人；
- 或者来自大家偷懒默认的那套（舆论、家人、强势一方）。

关系里的框架，决定：
- 谁在付出、谁在拿捏、谁在评判“对错”；
- 遇到冲突时，谁需要解释自己、谁只需要说“我就是这样”。

进化和社会心理角度：
- 群体中总要有人给出“这是正常的、这是不行的”的基调，其他人要么跟，要么走；
- 没有自己框架的人，会自动被归类到“跟随者/好用的人”，被安排在别人故事的边角。

## 2. 现实中的残酷表现

1）你以为在谈恋爱，她以为在“先看看”  
- 状态：你按男友标准投入时间和情绪；她嘴上说“顺其自然，不要给关系贴标签”。  
- 结果：你一提承诺就是“你太着急”，她一提要求你就得改。  
- 机制：她设定的框架是“我在体验，你在服务”，你从没谈过，只是默默接受。

2）你以为可以慢慢来，她默认你随叫随到  
- 状态：你刚认识不久就开始给 24 小时在线服务。  
- 行为：她随时找你排解情绪、处理麻烦；你有需求时她永远很忙。  
- 结果：在她框架里，你是“情绪管家”，不是平等关系对象。  
- 机制：一开始你就用行为告诉她“你生活绕她转”，她自然按照这个框架对待你。

3）职场里的“老黄牛框架”  
- 状态：你从入职起就加班、救火、不抱怨。  
- 行为：领导和同事默认你“抗压、能吃苦、不会说不”。  
- 结果：越累的活越多找你，升职加薪却轮不到你。  
- 机制：你设定的是“我只负责干”，别人给你安排的框架是“我只负责用”。

4）真正有框架的人  
- 状态：男D很清楚自己的底线和优先级。  
- 行为：遇到不尊重、不合理要求时，敢说“不”，也敢直接退出。  
- 结果：有人觉得他难搞，有人因此更尊重他；留下来的往往是适合他的那群人。  
- 机制：他的框架在不断筛人，而不是被人筛。

## 3. 执行 / 防御 / 利用方法

1）场景：关系刚开始时  
   动作：用“行为+少量明确表达”示范你的框架：
   - 行为：不 24 小时秒回，有自己的日程和优先级；
   - 表达：在轻松的聊天中提到“我很看重××，不接受××”。  
   台词：「我这边节奏是这样的：白天基本在忙，重要的事可以说清楚，我们一起协调。」  
   预期效果：一开始就让对方知道你有自己的轴，而不是一块任意捏的橡皮泥。

2）场景：遇到让你很不舒服的行为  
   动作：不急着哄、不急着解释，先把框架说清：
   - 描述行为+你的感受+你的边界；
   - 明确之后类似情况你的处理方式。  
   台词：「吵架可以，但骂人格我不接受。以后如果发生这种情况，我会直接结束对话，等都冷静再说。」  
   预期效果：对方知道你的底线是现实会执行的，而不是威胁。

3）场景：对方试图把你塞进他/她的剧本  
   动作：识别典型控制框架：
   - 「如果你爱我，你就应该…」
   - 「男人就该/女人就该…」
   - 「只有渣男才会拒绝我这个要求。」  
   操作：
   - 拒绝在这些前提下讨论，改回“我–你–事实”的框架。  
   台词：「我们可以讨论这件事对彼此的影响，但不是在‘如果你爱我就必须…’这个前提下。」  
   预期效果：不被情绪绑架，维持对自己行为的定义权。

4）场景：你想用框架筛选关系，而不是控制关系  
   动作：
   - 事先写下三条不可妥协的价值观/边界（比如忠诚、尊重、基本诚实）；
   - 在相处中遇到触碰底线的行为，给一次清晰表达机会，再犯就退出。  
   台词：「这不是我能接受的相处方式，如果这是你坚持的风格，那我们不合适。」  
   预期效果：框架成为筛选器，让不适合你的人早离场。

5）场景：你自己容易为留人不断退让  
   动作：
   - 每退一次一步，就在记事本上写下“我为留住TA牺牲了什么”；
   - 一旦发现你已经背叛了自己的核心规则，就强制停下所有进一步妥协。  
   台词：「如果为了这段关系我连自己都不要了，那这段关系已经没意义了。」  
   预期效果：防止在“怕失去”的情绪下把框架压到地板以下。

6）场景：你需要在团队或家庭里设框架  
   动作：
   - 用“规则+原因+后果”的方式说明，而不是简单拍桌；
   - 自己先严格执行，再要求别人执行。  
   台词：「晚上十点后不再聊工作，是为了第二天都能有状态。如果非要临时改，那我们要调整白天安排。」  
   预期效果：框架被视为“对系统有用的规则”，而不是你个人爱发号施令。

## 4. 常见陷阱与反噬

1）行为/心态：把框架当压人工具  
   为什么会踩：用“男人就该”“女朋友就必须”来偷换价值观。  
   反噬：短期也许能压住，长期只会积怨，最后所有人一起炸。

2）行为/心态：嘴上讲框架，行为全是跪  
   为什么会踩：说狠话比真执行容易。  
   反噬：对方会把你当成“只会嘴硬的纸老虎”，再也不会把你的话当真。

3）行为/心态：为了不孤单，一次次缩小框架  
   为什么会踩：宁可委屈自己也不想失去。  
   反噬：你失去的不是对方，而是自尊和底气。

4）行为/心态：试图在完全不成熟的人身上玩框架  
   为什么会踩：觉得自己能“教会对方”。  
   反噬：对方根本不想进你的游戏，你只是在和幻觉谈判。

5）行为/心态：以为“有框架=无敌”  
   为什么会踩：把框架神话成万能钥匙。  
   反噬：没底层价值支撑的框架只是空壳，强撑只会暴露虚弱。

## 5. 进阶阅读与验证

- 书籍：
  - 边界与成熟关系：《界限》《亲密关系》等；
  - 自我决定、自尊相关心理学书籍。  
- 论文方向：检索“relationship power”“personal boundaries”“assertiveness training”。

自我验证：  
1）写出你在亲密关系里最重要的3条规则：你能接受什么、绝不能接受什么。  
2）回顾过去两段关系，标记你在哪些场景违背了这些规则，是如何一步步退让的。  
3）下一段关系开始时，把这三条写在手机备忘录里，每次想妥协时对照一下：这是策略调整，还是核心自我被踩。  
4）当你开始习惯“宁可失去关系也不背叛自己”时，你的框架才算真正成型，而不是嘴上说说。
