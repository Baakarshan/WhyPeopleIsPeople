# Nash均衡

**核心一句话总结**：你们最后活成什么鬼样子，就是一套“谁都懒得先改变”的烂平衡，你不动，她也不会动。

## 1. 底层原理

Nash 均衡：在一个博弈里，当别人策略不变时，你单方面改变策略不会变得更好，这个状态就叫“均衡”。

放进亲密关系：

- 你习惯退一步，她习惯再进一步
- 你习惯沉默，她习惯用情绪和冷暴力逼迫
- 你习惯全部承担，她习惯全部甩锅

只要双方都认为“这样虽然难受，但改成本更高”，你们就会被锁在这种破局里——这就是你们的 Nash 均衡。

进化视角：

- 人脑天生懒，倾向于选择“稳定但不爽”的熟悉模式，而不是“可能更好但不确定”的新模式
- 长期关系中的一言一行，都会成为对方大脑的训练数据：
  - 你每次退让，她就学会“只要我坚持，他就会退”
  - 她每次冷暴力，你都去哄，她就学会“冷暴力是有效策略”

所以：

- 你们今天的相处模式，不是天上掉下来的，而是一点一滴互动训练出来的
- 只骂对方“怎么这样”，不承认自己也是这套均衡的一部分，就永远打不破局

## 2. 现实中的残酷表现

1）冷暴力-讨好型均衡
- 状态：她一生气就拉黑、失联、阴阳怪气，你一有风吹草动就先低头、求和。
- 行为：每次冲突都按这个剧本演一遍。
- 结果：她越来越不会好好说话，你越来越不敢提出自己需要。
- 机制：你们共同训练出一种均衡——“她用冷暴力，你负责修复”，谁都没动力换脚本。

2）暴吵-逃避型均衡
- 状态：一有问题，她直接炸，你直接躲。
- 行为：她摔东西、哭闹、放狠话，你要么转身走人，要么躲进手机和工作里。
- 结果：问题从不解决，只是周期性爆炸；她觉得“只有我发疯才有人理我”，你觉得“只要我躲，就不用面对”。
- 机制：在她大脑里，“发疯=有反馈”；在你大脑里，“逃避=短期减压”，于是这套互相折磨成了稳定状态。

3）烂婚姻里的“互相将就”
- 状态：两个人都知道这段关系没救，但都有各自不想失去的东西（房子、孩子、面子）。
- 行为：表面维持，私下各玩各的，偶尔爆炸再归位。
- 结果：谁都不开心，谁也不走。
- 机制：继续烂下去的痛苦 < 各自想象中“离婚的代价”，于是烂局反而成了均衡。

4）你一个人拼命修，ta一点都不改
- 状态：你看了书、上了课、学了沟通技巧，只在自己身上做功课。
- 行为：每次冲突你都更温柔、更懂事，她却一点不自省。
- 结果：你越来越累，她越来越理所当然。
- 机制：你单方面改变，却没改“她付出代价的结构”，均衡只是从烂→更烂，你一个人扛全场。

## 3. 操控 / 防御 / 利用方法

1）场景：想看清现在的烂均衡到底长什么样
- 动作：写出你们遇到问题时各自“标准反应三件套”。
- 台词（自问）： “她一不爽会做哪三件事？我一不爽会做哪三件事？结果通常是什么？”
- 预期效果：让你看到关系不是随机的，而是有一套固定策略组合，你才知道从哪一点下手调。

2）场景：你准备先改变，但不想变成一个人表演
- 动作：挑一个你能承受的策略变化，坚持一段时间，同时明确告诉对方你的新玩法。
- 台词：“以后吵架我不会再一味道歉，我会说清楚哪里是我能改的，哪里不能。你要是不接受，我们就冷静一阵各想各的。”
- 预期效果：打破过去“他必定先认错”的预期，让对方意识到旧套路不再稳定有效。

3）场景：对方完全不愿配合，只想维持既得利益
- 动作：不要继续做一个人“修复机器”，而是开始评估这段关系有没有改的价值。
- 台词：“我这段时间有在调整，但我感觉只有我一个人在动。你想不想一起调整？不想的话我们就得考虑要不要继续了。”
- 预期效果：把问题从“如何哄好她”升级为“这段关系值不值得救”，给自己一个退出选项。

4）场景：你们都意识到问题，但谁都拖着不谈
- 动作：约一个专门的时间谈“相处模式”，不在吵架时谈。
- 台词：“我们别等吵架才说这些。找个晚上，只聊我们平时各自的习惯+怎么调，你说一条，我说一条。”
- 预期效果：从情绪战场拉回思考区，一起在冷静状态下调整“均衡点”，而不是在情绪高涨时互相按头。

5）场景：你发现均衡已经烂到不值得救
- 动作：停止一切单方面修复，开始规划自己的退出路线。
- 台词（对自己说）：“我不是要把这段关系修到完美，而是要把自己从烂均衡里拉出来。”
- 预期效果：从“我要改变她”转为“我要抽身”，减少无效投入，把精力转移到能长期增值的地方。

## 4. 常见陷阱与反噬

1）幻想“不改变对方，就靠自己忍”也能迎来新局面
- 行为/心态：一味压抑和忍让，以为“总有一天她会感受到我的好”。
- 为什么会踩：害怕冲突，害怕失去。
- 反噬：你其实在训练她“这样对我也没事”，均衡只会越来越烂。

2）试图靠一次大爆发打破多年的均衡
- 行为/心态：平时不说，一次吵翻天，把所有老账一起翻出。
- 为什么会踩：不敢日常表达，只敢在情绪失控时一次性炸开。
- 反噬：对方只会记住“你很疯”，不会真正理解你要的改变，战后双方回到旧模式，外加一个“你脾气很差”的标签。

3）以为“谁先改变谁就输了”
- 行为/心态：死守自己烂策略，只因为不想让对方觉得“你服软了”。
- 为什么会踩：把自尊绑在错误的地方。
- 反噬：你们谁都不愿承担改的代价，最后一起在烂均衡里耗命。

4）装看不见对方的拒绝改变
- 行为/心态：对方一次次表明“不想、不愿、不改”，你却在心里替她找理由。
- 为什么会踩：承认她不想改，就等于承认这段关系的未来很差。
- 反噬：你用幻想维持关系，现实在用一天一天的消耗弄死你。

5）完全否定自己在均衡里的责任
- 行为/心态：所有问题都怪对方，自己永远是受害者。
- 为什么会踩：受害者身份最省力，不用做选择。
- 反噬：你失去改变局面的主动权，注定在下一段关系里复制同一套烂剧本。

## 5. 进阶阅读与验证

- 推荐阅读：
  - 《思考，快与慢》：理解习惯性反应和系统1/系统2的博弈。
  - 《亲密关系》：看长期关系如何形成稳定模式。
  - 博弈论入门书（如《策略思维》）：理解均衡和策略调整。

- 自我验证思路：
  1）写出你当前关系中“一吵架就会出现的三件事”，分别是谁做的，这是你们的默认脚本。
  2）选一个你能承担的策略改变（比如：不再秒回、不再一吵就道歉），连续做一个月，观察对方行为是否随之变化。
  3）和一位信任的朋友或咨询师梳理：你在关系里最典型的应对方式，是怎么在原生家庭中被训练出来的。
  4）如果你发现所有关系最终都稳定在“同一种烂均衡”，那就不是伴侣问题，而是你自己的策略系统问题，先修这个再谈恋爱。
